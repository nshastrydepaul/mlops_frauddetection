# Docs Directory

Store project documentation built with MkDocs here.

## Structure

- **`source/`** — Markdown documentation source files
- **`mkdocs.yml`** — MkDocs configuration (in project root)

## Usage

```bash
# Preview documentation locally
mkdocs serve

# Build static HTML
mkdocs build
```

## Guidelines

- Write clear, concise documentation for users and developers
- Include setup instructions, API docs, and usage examples
- Update docs alongside code changes
